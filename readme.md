## pdf to text program

### USAGE

1. with terminal
```zsh
cd pdf_to_txt
# open main.py
# __name__ == __main__ part sentence activate
# write path where you want and then
python main.py
```
2. with pyqt5
```zsh
cd pdf_to_txt
python run_layout.py
# write path where you want
# click convert
```
3. with standalone
```zsh
cd pdf_to_txt
pyinstaller -F run_layout.py
cd dist
./run_layout
# write path where you want
# click convert
```
